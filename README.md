# Brain Tumor Detection System

A CNN-based Brain Tumor Detection web application using TensorFlow and Streamlit.

# Features
- MRI image upload
- Tumor / No Tumor prediction
- Severity estimation
- Simple web interface

#Tech Stack
- Python
- TensorFlow / Keras
- Streamlit
- NumPy
- Pillow

#How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
